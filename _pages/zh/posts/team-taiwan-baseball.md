  <h2 style="color: #93c5fd; margin-top: 0; font-size: 1.6em;">寫給台灣棒球的一封情書</h2>
  <p style="font-size: 1.1em; line-height: 1.8; color: #f5f5f5;">
    棒球帶給人的感動很獨特——比賽中一局一局堆疊起來的張力、關鍵時刻爆發的情緒、還有全島一起屏息以待的那種心跳聲。對我來說，這份感動在<strong style="color: #fbbf24;">2024 年世界12強錦標賽</strong>台灣奪冠的那一刻，徹底成形。
  </p>
</div>

## 世界12強的那一刻

<div style="background: #fffbeb; border: 2px solid #f59e0b; border-left: 6px solid #f59e0b; padding: 20px; border-radius: 10px; margin: 20px 0;">
  <h3 style="color: #92400e; margin-top: 0; margin-bottom: 8px;">2024 世界12強——冠軍</h3>
  <p style="margin: 0; color: #666; font-size: 0.95em; line-height: 1.7;">
    最後一個出局數落定的那一刻，我到現在還記得自己在哪裡。台灣——一個大多數人在地圖上指不出來的島嶼——站上了國際棒球的頂點。那不只是一座獎盃，而是多年來的耕耘、熱情與堅韌的結晶。對於從小打棒球、在清大帶過壘球與棒球隊的我來說，那座冠軍的份量格外不同，感覺像是自己的事。
  </p>
  <p style="margin: 12px 0 0; color: #92400e; font-style: italic; font-size: 0.95em;">
    那場比賽在我心裡種下了一顆種子：如果我們能在最大的舞台上也做到呢？
  </p>
</div>

## 2026 WBC：讓我們驕傲的比賽

2026 年世界棒球經典賽，台灣最終沒有晉級八強——這是事實，我不會粉飾太平。但有兩場比賽，讓我想起自己為什麼會愛上這項運動。

<div style="display: flex; flex-wrap: wrap; gap: 18px; margin: 20px 0;">
  <div style="flex: 1; min-width: 280px; background: #ffffff; border: 2px solid #2563eb; border-left: 6px solid #2563eb; padding: 20px; border-radius: 10px;">
    <h3 style="color: #1e3a5f; margin-top: 0; margin-bottom: 8px;">台灣 vs. 捷克</h3>
    <p style="margin: 0; color: #666; font-size: 0.95em; line-height: 1.7;">
      這場比賽是那種會讓人從此愛上棒球的比賽。面對上屆 WBC 就讓世界驚豔的捷克，台灣帶著十足的拚勁與氣勢上場，球場裡的加油聲讓人覺得不管身在哪裡，都像是在主場。
    </p>
  </div>
  <div style="flex: 1; min-width: 280px; background: #ffffff; border: 2px solid #ef4444; border-left: 6px solid #ef4444; padding: 20px; border-radius: 10px;">
    <h3 style="color: #7f1d1d; margin-top: 0; margin-bottom: 8px;">台灣 vs. 韓國</h3>
    <p style="margin: 0; color: #666; font-size: 0.95em; line-height: 1.7;">
      讓我到現在還會起雞皮疙瘩的一場比賽。世仇對決、全場沸騰、每一球都是壓力。每一個打席都承載著兩國棒球數十年來的較勁與恩怨。不論勝負，這就是國際棒球存在的意義。
    </p>
  </div>
</div>

## 放眼未來

<div style="background: #f0fdf4; border: 2px solid #22c55e; border-left: 6px solid #22c55e; padding: 20px; border-radius: 10px; margin: 20px 0;">
  <h3 style="color: #15803d; margin-top: 0; margin-bottom: 8px;">台灣隊的未來</h3>
  <p style="margin: 0 0 12px; color: #666; font-size: 0.95em; line-height: 1.7;">
    這次沒能打進八強，現實就是這樣，心裡確實不好受。但我知道：人才在，熱情在，基礎建設也在進步。世界12強的冠軍證明了台灣在關鍵時刻有能力贏球，WBC 證明了我們的選手在任何一天都有和任何對手一較高下的實力。
  </p>
  <p style="margin: 0 0 12px; color: #666; font-size: 0.95em; line-height: 1.7;">
    我相信台灣隊終將闖進 WBC 八強——甚至走得更遠。這不是盲目的樂觀，而是跟兩好球之後還站在打擊區裡的信念一樣：<em>下一球，什麼都有可能改變。</em>
  </p>
  <p style="margin: 0; color: #15803d; font-weight: 600; font-size: 1em;">
    身為統計學家，我懂機率。身為棒球迷，我知道機率就是拿來被打破的。
  </p>
</div>
