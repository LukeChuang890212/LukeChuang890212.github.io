## 大学院での研究

<p style="color: #888; font-size: 0.9em; margin-bottom: 15px;">国立清華大学 統計学研究所</p>

<div style="background: #f8fafc; border-left: 5px solid #2563eb; padding: 20px; border-radius: 8px; margin: 20px 0;">
  <h3 style="color: #1e3a5f; margin-top: 0;">Ensemble-Based Multiply Robust Methods for Missing Not at Random Data with Model Misspecification</h3>
  <p style="margin: 0 0 8px; color: #888; font-size: 0.9em;">2023年春 – 現在 · 指導教員：鄭又仁教授</p>
  <p style="color: #475569; line-height: 1.7; margin-bottom: 10px;">
    Missing Not at Random データに対する新たなアンサンブル手法を提案した。複数の propensity score モデルと一般化モーメント法を組み合わせることで、multiply robust な逆確率重み付け推定量を構築し、軽度のモデル誤特定が存在するシナリオにおいてもその有効性をシミュレーション研究およびメンタルヘルスデータへの実データ適用を通じて実証した。
  </p>
  <span style="display: inline-block; background: #dbeafe; color: #1e40af; padding: 4px 12px; border-radius: 20px; font-size: 0.85em; font-weight: bold;">論文執筆中</span>
</div>

## 学部での研究

<div style="background: #ffffff; border: 2px solid #8b5cf6; border-left: 6px solid #8b5cf6; padding: 20px; border-radius: 10px; margin-bottom: 18px;">
  <h3 style="color: #4c1d95; margin-top: 0; margin-bottom: 6px;">VR 訓練が大学野球選手の投球パフォーマンスとレジリエンスに与える影響</h3>
  <p style="margin: 0 0 8px; color: #888; font-size: 0.9em;">2021年春 – 2022年春 · 指導教員：鄭匡佑（国立成功大学 体育健康休閒研究所）</p>
  <p style="margin: 0; color: #666; font-size: 0.95em; line-height: 1.7;">独自に開発した VR 訓練システムを用いて、プレッシャーの異なる状況が大学野球選手の投球パフォーマンスに及ぼす影響を調査した。あわせて、VR 技術がプレッシャー場面における選手のレジリエンス向上に寄与する可能性を検討した。</p>
</div>

<div style="background: #ffffff; border: 2px solid #f59e0b; border-left: 6px solid #f59e0b; padding: 20px; border-radius: 10px; margin-bottom: 18px;">
  <h3 style="color: #92400e; margin-top: 0; margin-bottom: 6px;">CPBL 観客来場意欲に影響する要因の分析</h3>
  <p style="margin: 0 0 8px; color: #888; font-size: 0.9em;">2020年 · 指導教員：黃柏僩</p>
  <p style="margin: 0; color: #666; font-size: 0.95em; line-height: 1.7;">対戦カードや気象条件といった要因が中華職業棒球大聯盟（CPBL）の観客来場意欲にどのような影響を与えるかを、興行成績を代理変数として分析した。データは CPBL 公式サイトおよび台湾中央気象局から取得した。</p>
</div>
